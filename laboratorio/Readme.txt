Instrucciones para instalación
1. Ejecute el php_action/config/script.sql para generar un schema con el nombre "prueba_unir" y una tabla personas.
2. Modifique el archivo de propiedades ubicado en la ruta php_action/config/db.ini según corresponda.
3. Coloque el directorio completo en la ruta de XAMPP o de PHP
4. Eliminar el php_action/config/script.sql una vez terminado el proceso.
